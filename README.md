# 🧠 Alzheimer Risk Analysis

A machine learning-based project that predicts the risk of Alzheimer’s disease using patient data. This project aims to assist in early detection by analyzing key health indicators and providing predictive insights.

---

##  Features
-  Data analysis and preprocessing of Alzheimer dataset  
-  Machine Learning model for risk prediction  
-  Simple web interface for user interaction  
-  Accurate and efficient prediction system  

---

##  Tech Stack
- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **Model:** Machine Learning (Pickle model)  
- **Frontend:** HTML, CSS  
- **Backend:** Flask

##  Installation & Setup

### 1️. Clone the repository
```bash
git clone https://github.com/fatimawaqar/Alzheimer-risk-analysis.git
cd Alzheimer-risk-analysis
```

### 2️. Install dependencies
```bash
pip install -r scripts/requirements.txt
```

### 3️. Run the application
```bash
python scripts/app.py
```

### 4️. Open in browser
```
http://127.0.0.1:5000/
```

##  How It Works
- The dataset is preprocessed and cleaned  
- A machine learning model is trained on relevant features  
- The trained model predicts Alzheimer risk based on user input  
- Results are displayed through a web interface
